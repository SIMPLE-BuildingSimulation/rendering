Fields

triangles: Vec<Triangle>

The Triangles in the scene that are not tested directly for shadow (e.g., non-luminous objects and diffuse light)

normals: Vec<(Vector3D, Vector3D, Vector3D)>

The normal of each vertex of each triangle.

front_material_indexes: Vec<usize>back_material_indexes: Vec<usize>materials: Vec<Material>

The materials in the scene

lights: Vec<Object>

A vector of Light objects that are considered sources of direct light. The objects here are also in the objects part.

distant_lights: Vec<Object>

A vector of distant Light objects that are considered sources of direct light

accelerator: Option<BoundingVolumeTree>

The acceleration structure that helps trace rays.

This needs to be build through the build_accelerator function.

sky_colour: Option<Spectrum<{ crate::N_CHANNELS }>>

The colour of the sky, normalized

sky: Option<Box<dyn Fn(Vector3D) -> Float + Sync>>

A function returning the diffuse Sky brightness (i.e., without the sun) The sun should be added separately. Alternatively, you can use the add_perez_sky function
